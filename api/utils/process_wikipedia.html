utils.process_wikipedia.title_to_id(page_title, path_to_db, lower=False) → Optional[str]

This function is adapted from https://github.com/jcklie/wikimapper Given a Wikipedia page title, returns the corresponding Wikidata ID. The page title is the last part of a Wikipedia url unescaped and spaces replaced by underscores , e.g. for https://en.wikipedia.org/wiki/Fermat%27s_Last_Theorem, the title would be Fermat’s_Last_Theorem. :param path_to_db: The path to the wikidata2wikipedia db :param page_title: The page title of the Wikipedia entry, e.g. Manatee.

Returns

If a mapping could be found for wiki_page_title, then return

it, else return None.

Return type

Optional[str]
