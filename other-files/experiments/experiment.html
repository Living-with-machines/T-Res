class experiments.experiment.Experiment(dataset: Literal['lwm', 'hipe'], data_path: str, results_path: str, dataset_df: pandas.core.frame.DataFrame, myner: geoparser.recogniser.Recogniser, myranker: geoparser.ranking.Ranker, mylinker: geoparser.linking.Linker, overwrite_processing: Optional[bool] = True, processed_data: Optional[dict] = {}, test_split: Optional[str] = '', rel_experiments: Optional[bool] = False, end_to_end_eval: Optional[bool] = False)ÔÉÅ

Bases: object

A class to represent an an entity linking experiment using NER, candidate ranking, and linking methods.

Parameters

  • dataset ("lwm", "hipe") ‚Äì The dataset to use for the experiment, must be set to either "lwm" or "hipe".

  • data_path (str) ‚Äì The path to the dataset directory (with processed data).

  • results_path (str) ‚Äì The path to the directory where the results will be stored. If it does not exist, it will be created.

  • dataset_df (pandas.DataFrame) ‚Äì The dataframe representing the resulting, preprocessed, dataset.

  • myner (recogniser.Recogniser) ‚Äì An instance of the NER model to use.

  • myranker (ranking.Ranker) ‚Äì An instance of the candidate ranking model to use.

  • mylinker (linking.Linker) ‚Äì An instance of the linking model to use.

  • overwrite_processing (bool, optional) ‚Äì Whether to overwrite the processed data if it already exists (default is True).

  • processed_data (dict, optional) ‚Äì A dictionary to store the processed data (default is an empty dictionary).

  • split (test) ‚Äì The data split to use for testing (train/ dev/test, default is an empty string).

  • rel_experiments (bool, optional) ‚Äì Whether to run end-to-end REL experiments (default is False).

  • end_to_end_eval (bool, optional) ‚Äì Whether to run the experiment for end-to-end evaluation or entity-linking-only evaluation (default is False, i.e. run EL-only).

linking_experiments() → NoneÔÉÅ

Run entity linking experiments on the processed data.

This function performs the entity linking experiments using the prepared data according to the different configurations of the recogniser, ranker and linker. The experiments are performed on different data splits and store the results in the specified format required by the HIPE scorer. Additionally, it provides an option to run end-to-end REL experiments.

Returns

None.

Note

The results of the experiments are stored in the self.processed_data attribute of the Experiment instance.
